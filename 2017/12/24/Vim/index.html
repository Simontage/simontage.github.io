Vim 备忘录

这真的是最后一次整理 VIM

Posted by Manson on 2017-12-24

前言

Vim是 Vi 的增强版，基本预装在大部分 Linux 发行版中。Vim 是一种全屏幕、多模式文本编辑器。这篇备忘录从开始进入编辑器到退出保存这一过程来梳理 Vim 的基本操作，包括 Vim 的模式 ，插入、复制删除和粘贴、替换查找文本等。

进入编辑器

方法一：

直接在当前终端键入 vim

方法二：

在终端键入vim file_name1使用 vim打开指定的文件。当指定的文件存在时，那么打开指定的文件；当指定的文件不存在时，作为新文件打开，即在当前目录中新建此文件。

Vim 的模式

Vim 有三种常用模式，分别是 Normal-mode、command-mode、Insert-mode

  • Normal-mode：正常模式，也是进入编辑器后的默认模式，此模式用来查看、复制和粘贴文本等操作。
  • Command-mode：命令模式，在命令模式可以执行 vim 内置的各种命令，也可以执行 shell 命令。
  • Insert-mode：插入模式，在插入模式下，对文本进行修改，如插入新的内容等。

三种模式之间互相切换

编辑操作

进入 Vim 编辑器后，那么编辑操作就开始了，进入 Vim 编辑器后默认的模式是 Normal-mode

Normal 模式下的编辑操作-移动、复制粘贴和删除

假设当前打开的文件是已经存在的文件。Normal 模式下的操作有：光标移动、复制粘贴删除、替换等

1.光标移动

光标移动有字符移动、单词移动、行内和行间移动、段落间移动

  • 字符移动

    在 Normal 模式下，使用 h、j、k、l 键移动光标。可以使用数字加上方向键(hjkl)来跳转多个字符。

  • 单词移动

    在 Normal 模式下，使用 w、b、e 以单词为单位移动光标。单词移动要比字符移动快捷一些，刻意练习单词移动，用来提高效率。

    w (word) ，1个 w 就向后移动一个单词，光标落在下个单词的词首；

    e (end) ，1个 e 向后移动一个单词，光标落在词尾；

    b (back) ，一个 b 就 向前移动一个单词，光标落在词首。

  • 行内和行间移动、句子间移动和段落间移动

    在行内移动中，使用0跳转至绝对行首，使用^跳转到第一个非空白字符行首，使用$跳转至绝对行尾。使用f/F + 字符移动至指定的字符位置，f向从左至右跳转，F相反。

    在行间移动中，使用行号G或:行号跳转至指定行的非空行首，使用G跳转至最后一行的非空行首，使用gg跳转至首行非空行首。使用数字-或数字k将光标向上移动指定行数，使用数字+或9空格、9j将光标向下移动指定行数。

    在句子间移动中，使用)跳转至下一句，使用(跳转至上一句。在段落间移动中，使用}跳转至下一段落，使用{跳转至上一段落。句子和段落移动使用的频率相对较低。

    实际上还有更多的光标移动方法，只是使用的频率相对较低。

2.复制粘贴删除

  • 删除 - d

    使用x删除光标所在处的字符，使用数字x删除光标所在处向后的指定个字符；

    使用df 单个字符删除光标到指定字符间的文本。

    使用dd删除光标所在处的整行字符，删除的文本存入寄存器中，随后使用p可以粘贴删除的文本，使用数字dd删除光标所在行的指定行数

    使用d和各种光标定位符配合，如d0、d^、d$、dw、de、db等，可以删除当前光标至指定位置的文本

  • 替换/改变 - c/r

    使用cc删除光标当前所在行，并转为 Insert 模式，使用c$删除光标所在处至行尾，并转为 Insert 模式，使用数字cc删除指定数量的行后转为 Insert 模式。

    c命令也可以结合各种位置操作，参考 d

    使用r替换当前光标所在处的字符 ，使用R替换当前光标所在处以及之后的多个字符，直到Esc退出。

  • 复制 - y

    使用yy复制光标所处的一整行，使用数字yy复制光标所在的指定个数行的文本，使用y$复制当前光标所在出至绝对行尾的文本。

  • 粘贴和剪切

    当复制或删除的是整行时：

    粘贴到当前光标所在行的下方 ：(小写)p

    粘贴当前光标所在行的上方：（大写）P

    当复制或删除的不是整行文本时：

    粘贴到当前光标所在字符后：（大写）P

    粘贴到当前光标所在字符前：（小写）P

  • 撤销

    使用u撤销上一次操作，撤销多次操作使用数字u，默认可撤销的最大次数为50次。

    Ctrl+r是重做命令，也就是撤销此前的撤销操作。

4.查找

在 Normal 模式中，键入/keyword或?keyword，可以对全文进行查找操作，查找的结果高亮显示，使用n和N来查看查找结果。

关闭高亮使用:set nohlsearch，不过至对当前 shell 生效。查找也支持正则表达式。

Command 模式下的编辑操作

在 Normal 模式下，键入:即进入 Command 模式。

1.范围定界

在末行模式中，可以通过start,end来定位一个范围，如10,19或10,+9，都表示选择第10行到第19行的文本。在范围定界中，.表示光标所在的行，%表示全文，$表示文件末尾。也可以使用.,$-2，表示从当前行到文档末尾倒数第二行。

范围定界还可以使用模式\pat1\,\pat2\，表示从当前光标所在行第一次被模式1匹配到的行到被模式2匹配到的行。

2.查找替换

在末行模式中，使用:地址定界s/查找模式/替换后的内容/gi/进行查找并替换操作。查找模式是正则表达式，替换后的内容可以使用引用，但不可以使用模式。g 表示全局替换，即把每一行出现多处匹配的内容都进行替换；i 表示忽略大小写。

查找替换使用的分隔符，可以使用/也可以使用@或#等特殊字符。

?? (&) 如：将一个文本文件中非#开头的行加上#号：:1,30s/^[^#].*/#&/g
