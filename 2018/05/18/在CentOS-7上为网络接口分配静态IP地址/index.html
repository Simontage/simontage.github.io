在CentOS 7上为网络接口分配静态IP地址

Posted by Manson on 2018-05-18

相关服务

在 CentOS 7中，网络管理服务(NetworkManager.service)是一个动态网络控制器和配置系统，并默认安装启用。

通过systemctl status NetworkManager可以查看网络管理服务当前的运行状态；

使用nmcli dev status查看接受网络管理器管理的网络接口，输出结果是 connected，表示该接口受网络管理器管理。

方法二：使用 nmtui 修改

nmtui 是网络管理图形化文本配置工具，使用 nmtui 配置的前提是该网卡接受 NetworkManager 管理，因此在该网卡的配置文件中，NM_CONTROLLED应该配置为 yes。如果系统没有安装 nmtui，使用yum install NetworkManager-tui。使用 nmtui或 nmtui eidt 接口即可修改该接口的 IP 地址信息等。编辑完成后，重启网络服务。
