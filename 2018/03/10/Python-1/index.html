定义变量

变量的定义规则

  1. 变量名只能是字母、数字和下划线的任意组合

  2. 变量名的第一个字符不能是数字

  3. 以下关键字不能声明为变量名

    and as assert brreak class continue def del elif else except exec finaly for from global if import in is lamdba not or pass print rasie return try while with yield

最后print语句输出的是Adidas Nike，name2之所以还是Nike，是因为name2这个变量是指向name变量指向的内存地址，而不是指向name。

Python官方建议对变量的命名才是下划线定义变量，如name_of_shoes。另外还有驼峰法用来命变量，在Windows中更为常见。

在Python中，没有常量的概念，大写的变量名表示 常量，不应该被修改。

进制和编码

计算机通过0和1就能表示任意数字，是因为多个0与1这样的组合可以表示的可能性有多种，这就是二进制与十进制转换。

512 256 128 64 32 16 8 4 2 1

这里前面的数值是后面数值之和加一，也是2的0次方，开始2的1次方

计算机中使用二进制可以表示任意数字后，还需要表示英文字母和一些字符，因此ASCII码表规定了一个特定的二进制数值表示一个特定的字母、数字或字符。ASCII码使用8位二进制数值定义英语和西欧语言字母，实际上使用7位（0-127）就能将全部字符表示。

但是不同语言情况不同，因此产生了不同的字符集，如中文的GBK和GB10830等。当字符使用的字符集和系统设置的字符集不同时，就会无法显示正确的字符。因此Unicode来解决使用不同编码集导致乱码。Unicode使用16位（两字节）表示一个字符，这样对于英文等西欧语言来说，浪费了大量空间。UTF-8是可变长Unicode，也是Unicode子集，UTF-8存储英文等西欧语言时，采用的是ASCII编码集，存一个汉字需要3字节。

python2默认采用的是ASCII编码，如果Python编码中含有中文，则需要指明使用的字符编码集

python3默认使用的UTF-8。

注释使用

Python中单行注释使用#，多行注释以及多行内容使用'''... ...'''。在Python中双引号和单引号没有区别，只需要注意在双引号嵌套中，需要注意不同级别的要使用双引号和单引号区分开，即单引号中套双引号，或双引号中套单引号。
